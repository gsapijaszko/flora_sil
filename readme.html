Flora Siliesiaca

Historyczne dane flory lska bazujce na artykuach publikowanych w Jahres-Bericht der Schlesischen Gesellschaft fr vaterlndische Cultur. Elelktroniczne wersje sprawozda znajduj si w Opolskiej Bibliotece Cyfrowej oraz lskiej Bibliotece Cyfrowej

Pliki rdowe, podzielone latami, znajduj si w katalogu source. Struktura wpisu:

jahres <- data.frame(
  species = "Cystopteris fragilis",
  citation = "@schalowErgebnisseDurchforschungSchlesischen1934",
  entry = "Cystopteris fragilis Naumburg a. B.: Schlopark (Tscheppe)!; [...]",
  lon = 15.24453, 
  lat = 51.80259,
  comments = "Nowogrd Bobrzaski, park zamkowy (Wzgrze Zamkowe)"
) |> rbind(jahres)

Bibliografia dodana jest jako obiekt klasy bibentry przy uyciu pakietu RefmanageR

bib <- RefManageR::as.BibEntry(
  c(
    bibtype = "article",
    key  = "schalowErgebnisseDurchforschungSchlesischen1934",
    author = "Emil Schalow",
    title = "Ergebnisse der Durchforschung der schlesischen Gefsspflanzenwelt im Jahre 1933",
    journal = "Jahres-Bericht der Schlesischen Gesellschaft fr vaterlndische Cultur",
    date = "1934",
    volume = "106", 
    pages = "140--156")
)
